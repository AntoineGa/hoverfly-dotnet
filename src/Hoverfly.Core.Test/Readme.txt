NOTE:

This project doesn't include the hoverfly.exe
So to run the tests you need to download Hoverfly and add the hoverfly.exe
You can add the path to hoverfly.exe to your PATH environment variable in Windows.
Or you can add it to the root folder of the test project ouput filder or any subfolder 
you like under the output folder. You can also using the HoverflyConfig to specify 
the base path to the hoverfly.exe file.

You can download it from http://hoverfly.io/#download
